# PARTE 2: CONCEITOS DE SKILLS — ANÁLISE DETALHADA

## 2.1 Constelação de Habilidades 3D (Skill Galaxy)

**Inspiração:** `sunnypatell/react-threejs-portfolio` (⭐26) usa galáxia de partículas. GitHub Skills Graph. Neo4j Browser.

**No seu stack:**
```
R3F + drei (Stars, Line, Sphere) + d3-force-3d + OrbitControls
```

**Layout:** Nós coloridos por categoria (indigo/cyan/amber/rose). Conexões sutis entre skills relacionadas (Rails ↔ REST ↔ PostgreSQL). O usuário orbita com mouse.

**Por que funciona no seu caso:** Já tem tema "Connected Workspace". Constelação é a metáfora visual de "conexões". Cada skill é uma estrela — a proficiência determina o tamanho/brilho.

**Referências diretas:**
- Active Theory (`activetheory.net`): constelações de nodes 3D
- Lusion "Oryzo AI": orbes que se conectam (`lusion.co/projects/oryzo_ai`)
- Bruno Simon: objetos 3D com física que o usuário explora

## 2.2 Skill Tree RPG (Árvore de Talentos)

**Inspiração:** `codebucks27/Next.js-Creative-Portfolio-Website` (⭐386, tema wizard). Path of Exile Skill Tree.

**No seu stack:**
```
SVG/Canvas 2D + GSAP ScrollTrigger + CSS custom properties
```

**Layout:** Árvore com nodes hexagonais. Ramos conectam categorias. Scroll "desbloqueia" nodes com brilho dourado e efeito de glow. Cada node tem ícone SVG (já feito no Skills atual).

**Por que funciona:** Gamificação instantânea. O stack do codebucks27 (⭐386) é exatamente Next.js + Three.js + Tailwind + Framer Motion. Código aberto. Tutorial no YouTube.

## 2.3 Orbes Flutuantes 3D (Floating Orbs)

**Inspiração:** `Oryzo AI` (Lusion, DEV SOTD). Bruno Simon. Evolve Studio.

**No seu stack:**

```
R3F + MeshDistortMaterial + shaderMaterial com fresnel glow + @react-three/cannon
```

**Layout:** Esferas orbitando em anéis concêntricos. Tamanho = proficiência. Cor = categoria. Shaders com iridescência (20 linhas de GLSL). Ao clicar: esfera expande com tooltip.

## 2.4 Cidade Isométrica (Isometric Skill City)

# PARTE 3: CONCEITOS DE UNBOXING — ANÁLISE DETALHADA

## 3.1 Scroll-Driven Caixa 3D (The Box)

**Inspiração:** Apple Product Pages. `Lusion "Choo Choo World"`. `T11` (WOTD 8.41).

**Técnica:** R3F Box com tampa articulada. GSAP ScrollTrigger com `scrub: true` vincula abertura ao scroll. `MeshTransmissionMaterial` (glass morphism) na caixa. Conteúdo emerge via `Html` do drei ou overlay HTML absoluto.

## 3.2 Blueprint → Realidade (Architectural Unfold)

**Inspiração:** `Lusion "Devin AI"`. `Obys Agency`. Apple M-series chip reveals.

**Técnica:** Fase 1: wireframe azul (`EdgedGeometry` + `MeshBasicMaterial wireframe`). Fase 2: tween para material sólido + texturas. Para cada projeto: mockup de dashboard (Tradener), gráfico de analytics (Modulus), coordenadas 3D (IEEE — já tem point cloud).

## 3.3 Origami Card Unfold

**Inspiração:** `Petr Knoll "Off Menu"` (DEV SOTD). `Mr. Pandas Paper Portfolio` (DEV SOTD). Paper.js.

**Técnica:** CSS 3D transforms ou R3F Planes com animação de vértices. GSAP Flip plugin para estados dobrado/desdobrado. Sombra dinâmica.

## 3.4 Layered Peel (Casca de Cebola)

**Inspiração:** `Obys Agency`. `The Lookback` (Gil Huybrecht). `Studio Dialect` (DEV SOTD).

**Técnica:** `clip-path` animado ou `mask-image` com gradiente. Cada camada: textura diferente (papel kraft, foil metalizado, plástico glassmorphism). Scroll-triggered "rasgo".

## 3.5 Device Mockup Theatre

**Inspiração:** Apple Keynotes. `Lusion "Synthetic Human"`. `Ruinart Digital Fresco` (makemepulse).

**Técnica:** Modelos 3D de devices (Baixar do Sketchfab CC ou criar com Box + Cylinder). `useGLTF` do drei. Animação de montagem: base → tela → conteúdo na tela (`Html` overlay). Para Tradener: monitor ultrawide com dashboard. Para Modulus: laptop com analytics.

## 3.6 Shrink-Wrap Burst

**Inspiração:** `Gucci Vault`. `Max Mara Jacket Circle`. `Ruinart Digital Fresco`.

**Técnica:** Shader customizado com noise displacement (plástico enrugado). Mouse afeta distorção. Scroll-triggered tear via shader `discard` ou `clip-path`.
